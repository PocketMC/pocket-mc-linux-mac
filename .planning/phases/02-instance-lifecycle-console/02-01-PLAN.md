<threat_model>
### Secure Extraction Policies

- **T-02-01: Zip Slip Path Traversal**
  - *Severity:* High
  - *Threat:* Malicious ZIP archives containing relative path escapes (e.g., `../../etc/passwd` or outside the instances folder root) could overwrite system files or execute arbitrary code.
  - *Mitigation:* The zip extraction engine must canonicalize the destination path of each ZIP entry, check if it starts with the canonical path of the target instance directory, and immediately abort the extraction, delete all partially written files, and throw a path security exception if a traversal attempt is detected.
</threat_model>
