<threat_model>
### Process and Input Security Policies

- **T-02-02: Orphaned Server Processes (Resource Exhaustion)**
  - *Severity:* High
  - *Threat:* Crashed or stopped servers leaving orphaned background JVM or PHP child processes running, consuming CPU/RAM and preventing restart by locking ports.
  - *Mitigation:* The process runner must call `setpgid` immediately on process launch. If a graceful shutdown command or signal (SIGTERM) fails to exit the group within 15 seconds, the runner must issue `kill -9 -pgid` (SIGKILL) to terminate the entire process group.
</threat_model>
